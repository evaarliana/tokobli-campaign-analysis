# TokoBli Campaign Analysis

## Overview

This project analyzes promotional campaign performance and customer purchasing behavior for TokoBli, an e-commerce platform in Indonesia.

The analysis evaluates campaign effectiveness, identifies high-performing product categories, and measures the impact of a new product page through A/B testing.

## Objectives

* Evaluate campaign performance based on revenue, transaction volume, and discount spending.
* Identify the most efficient promotional strategy.
* Discover high-performing product categories.
* Assess the impact of a new product page design using A/B testing.
* Generate actionable business recommendations.

## Tools

* Microsoft Excel
* Google Sheets
* Descriptive Statistics
* A/B Testing
* Business Analysis

## Key Findings

* Campaign 11.11 generated the highest revenue among all campaigns.
* Campaign 12.12 achieved the best revenue-to-discount efficiency ratio.
* Men's Fashion was the highest revenue-generating category.
* Several product categories delivered strong performance with relatively low promotional spending.
* The new product page increased average transaction value compared to the previous design.
